# AI Tools Used

Used Claude and ChatGPT at a few points during the build.

React Context was one area where I took help — specifically wiring up the provider with localStorage sync and making sure state updates were propagating correctly to all pages. Used Claude to get that part right.

Also used ChatGPT for debugging — ran into a re-render issue in the order modal and some unexpected behaviour with useParams in React Router. Easier to just describe the error and get a fix than spend 30 minutes on it.

A few other smaller things where the implementation felt a bit tedious — like the form validation setup with React Hook Form — I used AI to speed things up.
