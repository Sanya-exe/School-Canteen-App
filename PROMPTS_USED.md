# Prompts and AI Tools Used

I used Claude (Claude Code by Anthropic) to help me understand and build this project since I'm still learning React. Here's roughly how I used it:

- Asked it to explain what the assignment was asking for in simple terms
- Got help understanding how React Context works and when to use it instead of just passing props around
- Used it to help set up TailwindCSS with Vite (the config part was a bit confusing at first)
- Asked for help structuring the folder layout before writing any code
- Got explanations for React Hook Form — specifically how `register`, `handleSubmit`, and `formState.errors` work together
- Used it to understand how `useParams` works in React Router when building the student detail page

I wrote the components myself based on understanding the concepts, and referred back to Claude when something wasn't working or I didn't understand why something was needed.
